Resonara is a professional audio workspace for macOS and Windows — transcode and loudnorm, play a hybrid sample piano, and speak long documents with offline neural engines (Piper, Kokoro, optional expressive). Fully local. No cloud account.

Three studios. One install.

Everything runs on your machine — ideal for privacy, offline work, and deterministic audio pipelines.

◈

Audio lab

Production-grade local processing powered by ffmpeg.

  • Import with magic-byte validation
  • Transcode MP3, AAC, FLAC, OGG, Opus, WAV
  • Two-pass EBU R128 loudnorm
  • Waveform, silence detect, trim & stream
♪

Hybrid piano

Sample-pack instrument with takes and export.

  • Play notes from the upright seed pack
  • Record and manage takes
  • Analysis hooks into the job pipeline
  • Export ready for further processing
◎

Voice (TTS)

Long-form speech with offline neural engines (Piper, Kokoro, optional expressive).

  • macOS say · Windows System.Speech
  • Automatic chunk → synthesize → concat
  • Built for multi-paragraph / 10k+ words
  • Progress without freezing the UI

From install to first export

Designed for people who want results without a terminal ritual.

Install

Download the macOS DMG or Windows setup. Drag to Applications or run the installer.

Launch

Resonara starts a local lite engine and opens the studio UI automatically.

Create

Process audio, play piano, or paste a long document into Voice — then export.

Built for local power

Electron shell · NestJS engine · ffmpeg · optional full Docker stack for servers.

Desktop lite mode

Default for end users. SQLite-compatible sql.js, filesystem object storage, and inline jobs — zero Docker.

RESONARA_LITE=1 Electron as Node Offline-first

Health you can trust

GET /health reports database, ffmpeg (with smart path resolution), and TTS engine status — so GUI apps still find Homebrew ffmpeg.

Download & platforms

Installers ship with offline Piper voices for English and Brazilian Portuguese. Unsigned v2.1 builds are fine for local use; right-click → Open on first macOS launch.

macOS

DMG for Apple Silicon. Requires ffmpeg on PATH (brew install ffmpeg). Bundled en + pt-BR neural voices.

Windows

NSIS x64 installer with Start Menu and desktop shortcuts. Bundled piper.exe + en/pt-BR models.
